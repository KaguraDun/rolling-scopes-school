Type Script : Introduction

by Vasilii Kovnev

What is it?

TypeScript is an open-source language which builds on JavaScript with adding static type definitions.

What is the benefit?

  • TypeScript supports static typing
  • TypeScript makes code easier to read and understand
  • TypeScript reduces the number of bugs
  • TypeScript encourages developers to follow OOP best practices
  • TypeScript saves developer time

Interfaces

          

            interface LabeledValue {
              label: string;
            }
            
            function printLabel(labeledObj: LabeledValue) {
              console.log(labeledObj.label);
            }
            
            let myObj = { size: 10, label: "Size 10 Object" };
            printLabel(myObj);

Writing the function type

          
            let myAdd: (x: number, y: number) => number = function (

              y: number
